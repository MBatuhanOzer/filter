# Image Filter Engine

This project is an Image Filter Engine that allows users to apply various filters to images. 
It supports multiple image formats and provides a user-friendly interface for selecting and applying filters.
You can use it by directly putting ``` src/filter-engine ``` folder to your project and calling function from ```filter.h```.
You need to call initialize filter_engine_start() before using any filter function.
## Features
- Currently supports the following filters:
  - Grayscale
  - Sepia
  - Invert Colors
